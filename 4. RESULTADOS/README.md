# 4. RESULTADOS

Esta carpeta contiene salidas analiticas ligeras, especialmente graficos.

Subcarpetas principales:

- `panel_diagnostico/`: visualizaciones del chequeo longitudinal de NORDEMP
- `descriptivos_exposicion/`: graficos exploratorios del analisis de exposicion

Convencion actual:

- Las tablas y bases derivadas se guardan en `1. DATOS/`.
- `4. RESULTADOS/` se reserva principalmente para figuras y salidas visuales.
